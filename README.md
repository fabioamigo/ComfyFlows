### Install

```
curl -LsSf https://raw.githubusercontent.com/fabioamigo/ComfyFlows/refs/heads/main/install.sh | sh
```

### Run

```
. .venv/bin/activate
cd ComfyUI
python main.py --listen 0.0.0.0 --port 9999
```
